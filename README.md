# Spring Boot Maven Project

## Description
This project is a simple Spring Boot web application created using Maven. It demonstrates project creation, dependency management, application execution, and GitHub integration.

## Technologies Used
- Java 21
- Spring Boot 4.1.0
- Apache Maven

## Features
- Responsive homepage
- Embedded Tomcat server
- Modern UI
- Maven build system

## How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `SpringbootDemoApplication.java`.
4. Open `http://localhost:8080`.
